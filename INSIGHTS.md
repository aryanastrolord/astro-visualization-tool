## Combat Concentrates in a Few Locations

Most kills happen in just a few map locations. The rest of the map barely sees any action.

**Why:** Players gravitate toward areas with loot, cover, and sightlines. Empty areas are ignored.

**Fix:** Add rare item spawns to unused zones. Redesign weak hotspots. Create a fifth strategic location with unique value (tower, bridge, underground passage).

**Measure:** Track kill distribution. Spread combat to more areas of the map.

---


## Storm Direction Is Predictable

Players use the same extraction routes every match. Other options are never chosen.

**Why:** Storm always shrinks toward the same direction. Players memorize the optimal path. No variation.

**Fix:** Randomize storm direction. Create multiple extraction points with different risk/reward tradeoffs (safe-but-far, close-but-hot, medium).

**Measure:** Track which extraction routes get used. Goal: Make multiple routes viable each match.

---

## Humans Are Outnumbered

Matches have way more bots than humans. Most kills are bot-on-bot. Humans barely interact with each other.

**Why:** Bots vastly outnumber humans. Most combat happens between bots, not with humans.

**Problem:** Humans feel like spectators watching bots fight instead of being the main event.

**Fix:** Reduce bot count per match. Cluster bots into waves so they encounter humans, not just each other.

**Measure:** Track how many kills involve humans. Goal: Make human-vs-something the primary engagement.

---

## What Needs to Change

- Spread combat across the map instead of 4 hotspots
- Make bots survive longer
- Add variety to storm patterns and extraction routes
- Reduce bot count so humans encounter each other more

Data source: Analysis of 79 matches using Astro Analytics.
